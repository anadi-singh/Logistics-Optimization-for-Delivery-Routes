# 🚚 Logistics Optimisation for Delivery Routes

## 📌 Project Overview

A SQL-based analytics project focused on improving delivery efficiency by analysing logistics data, warehouse performance, and transportation routes.

---

## 🎯 Business Problem

Delivery companies often struggle with:

- Delayed deliveries
- Inefficient routes
- High transportation costs
- Warehouse bottlenecks

This project demonstrates how SQL analytics can identify operational inefficiencies and support better logistics decisions.

---

## 🛠 Technologies Used

- SQL
- MySQL
- Data Analysis
- Database Design

---

## 🔄 Workflow

1. Data Cleaning
2. Database Design
3. SQL Queries
4. Performance Analysis
5. Business Insights

---

## 📊 Results

- Optimised delivery routes
- Identified warehouse bottlenecks
- Improved operational efficiency
- Reduced delivery delays

---

## 💡 Business Recommendations

- Optimise warehouse allocation
- Improve route planning
- Reduce idle delivery time
- Forecast delivery demand
